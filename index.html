HELLO WORLD

A Hello World App built with Flutter that introduces beginners to fundamental Flutter concepts. This app features a simple user interface with a button and a text display. Initially, it shows "Hello, World!" in English with a default background color. When the user clicks the button, the background color changes randomly, and the text updates to say "Hello" in different languages, such as "Hola" (Spanish), "Bonjour" (French), "Kumusta" (Filipino), and more. This interactive experience helps users learn about state management, button handling, and dynamic UI updates in Flutter.

TO-DO-LIST

A To-Do List App built with Flutter that helps users efficiently manage their daily tasks and boost productivity. The app allows users to create, edit, and delete tasks seamlessly through an intuitive and user-friendly interface. Users can also set due dates, prioritize tasks, and mark them as completed for better organization. To enhance user experience, the app supports persistent data storage so that tasks are saved even after closing the app. It also includes reminders and notifications to ensure that important tasks are never forgotten. With a sleek and responsive design, this app is perfect for students, professionals, and anyone looking to stay organized on the go

QUIZ RECORDER

The Quiz Recorder App is a Flutter-based application designed to help users efficiently track and manage their quiz scores over time. Whether you're a student, teacher, or self-learner, this app provides a structured way to monitor performance, analyze progress, and stay motivated throughout the learning journey. With a clean and intuitive interface, users can easily record quiz details, including the quiz title, date, score, and total possible score. The app ensures that all entries are neatly organized, making it effortless to review past quiz results whenever needed.
